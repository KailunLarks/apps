# 自然成陆期（唐-清代）

## 1. 地理演变

- **唐代**：南渡江泥沙在海口湾入海口大规模沉积，形成"海田岛"和"白沙门岛"雏形沙洲，形态极不稳定
- **明代**：沙洲逐渐靠拢连缀，岛屿轮廓初步显现。正德《琼台志》记载"有海岛五"，表明此时为多岛组成的群岛格局（包括今主体部分、白沙门及周边沙洲）
- **清代**：岛屿间水域最终被泥沙填满，连为一体

## 2. 地名演变

- **海田**：明代已有此名（万历《琼州府志》："番民所，在海口浦，即今海田村"），意为"海边的水田"，反映早期农业经济特征
- **海甸**：1887年张之洞借用朱元璋"南溟奇甸"典故，将"海田村"雅化为"海甸"，赋予文化内涵和战略地位
- **白沙门/白沙津**：因洁白沙滩得名，"津"指渡口港口

## 3. 港口与商贸军事

| 朝代 | 关键事件/制度 | 地位与影响 |
|------|--------------|-----------|
| 宋代 | 设"海口浦"；王光祖疏浚航道；淳祐年间飓风冲开航道得名"神应港" | 海上丝绸之路南海北部重要集散地 |
| 元代 | 设"白沙津镇"，派驻"白沙水军"常态化巡防 | 国家海防体系南海北部重要支点 |
| 明清 | 官渡渡口，与民间商贸"海口港"功能分化并存 | 连接海南与大陆的官方咽喉要道 |

## 4. 民间信仰体系

- **天后宫（妈祖庙）**：始建于元代，由福建商贾所建，海南妈祖文化起始地，承载海洋文化、移民文化和官方民间互动
- **"一寺八庙"**：海甸溪北岸不足三公里的沿江二路沿线，集佛教（仁心寺，传始建于唐）、儒家、道家及地方民俗信仰于一体，供奉关公、康公、鲁班、冼夫人等多元神祇组合

## 5. 疍家文化

- **生活方式**："以船为家，浮生江海"，世代以海为生，形成独特渔业文化、生产习俗和信仰
- **文化特征**：人与自然和谐共生的产物，体现对自然力量的敬畏与尊重

## 1. 1970年"拦海造田"工程

- **背景**：响应"农业学大寨"号召，向海要地扩大耕地面积
- **规模**：动员超100万人次劳动力，修筑11公里长防潮大堤，围垦6.5平方公里土地
- **影响**：短期扩大陆地面积，但长期破坏自然水系，导致水质恶化、泥沙淤积，生态代价沉重

## 2. 高校建设

- **1947年**：私立海南大学创办于沙尾椰子园，1950年撤销，开创海南现代高等教育先河
- **1983年**：新海南大学成立，选址海甸岛西北部滨海荒滩，由第一代建设者艰苦创业
- **2007年**：与华南热带农业大学合并，2008年进入国家"211工程"，成为海南省唯一"双一流"建设高校

## 3. 渔业文化变迁

- **衰落**：城市化推进和港口功能转移，传统渔业走向衰落，渔民上岸定居，年轻一代不继承祖业，渔业人口锐减
- **保护**：咸水歌、疍家婚俗等列入非物质文化遗产名录，但面临传承人断层、文化空间丧失等困难

## 4. 地名与集体记忆

- **古地名保留**：发展中有意识地保留"海田""白沙门"等古地名，维系居民情感和文化认同

## 1. 发展定位转变

- **核心理念**：从"征服自然"转向"与自然对话"，从外延扩张转向内涵提升
- **功能定位**：打造"人文休闲宜居岛"，集文化体验、休闲度假、生态游憩和高品质居住于一体
- **目标**：实现从"宜居"到"宜业""宜游"的全面提升

## 2. 生态修复实践

### 海甸溪综合整治

- **措施**：截污纳管、清淤疏浚、生态修复（种植水生植物、构建生态浮岛、放养滤食性鱼类）、景观提升
- **成效**：从"臭水沟"变为城市滨水绿带，体现"人水和谐"理念

## 3. 现代服务业与旅游业

- **商业发展**：大型超市、连锁餐饮、巨幕影院、购物广场入驻，形成多个商业圈。海甸溪北岸打造"天汇老码头风情商业街"（投资近10亿元），发展夜经济
- **旅游升级**：
  - 传统景点：白沙门公园、海口钟楼、骑楼老街等
  - 新业态："旅游+"模式深化，如"海南大院"（非遗文化+餐饮）、华彩·海口湾广场（品牌买手店、潮牌集合店）等满足年轻消费群体

## 4. 文化传承与活化

- **庙宇文化延续**：保留"一半海水一半庙"特色，天后宫、"一寺八庙"、关康庙等构成多元信仰画卷
- **历史文脉保护**：将"海田"文化、"拦海造田"精神、渔村风情融入城市景观和公共空间，形成独特文化标识
